## Instruções de Instalação e Execução

**Requisitos:**
- [Node.js](https://nodejs.org/en)
- [NPM](https://www.npmjs.com/)
- [mkcert](https://github.com/FiloSottile/mkcert)

### Terminal 1

```
cd project

npm install
npm run install:components

mkcert -install
mkcert localhost

npm run dev
```
